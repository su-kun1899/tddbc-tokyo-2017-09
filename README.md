# tddbc-tokyo-2017-09

## Overview

[TDDBC Tokyo 2017-09](https://tddbc.connpass.com/event/65484/) で使用するために作成したプロジェクトです 

プロダクトコードをJava, テストコードをSpockで書くことを想定しています。

ビルドはMavenで行います。

## How to run / test

```
./mvnw clean test
```
